## Overview

Computer vision-based robot assistant that detects and picks nuts using YOLOv8 and a collaborative robot arm. Responsible for robot motion, simulation, model training, and project management.

## Role

- Robot motion planning & control
- Isaac Sim simulation setup
- YOLOv8 model training
- Project Manager
