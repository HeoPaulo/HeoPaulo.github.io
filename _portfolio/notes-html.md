## Overview

A feature-rich, single-file web application for personal note-taking. Includes real-time Supabase cloud sync, a Markdown editor with syntax highlighting, dark/light theme toggle, search, calendar, and checklists — all in one standalone HTML file with no build step required.

<!-- ![Screenshot]({{ site.baseurl }}/assets/img/projects/notes/screenshot.png) -->

## Features

- Real-time cloud sync via Supabase
- Markdown editor with syntax highlighting
- Calendar view & checklists
- Dark / Light theme toggle
- Zero build step — single HTML file
