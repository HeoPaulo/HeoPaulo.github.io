## Overview

Digital twin platform for robotic process automation simulation. Built CAD models and integrated simulation environments to mirror real-world robot behavior for testing and validation.

<!-- ![Simulation]({{ site.baseurl }}/assets/img/projects/digital-twin/sim.gif) -->

## Key Features

- CAD model integration with simulation
- Real-time synchronization between physical and virtual robot
- Validation and testing environment
