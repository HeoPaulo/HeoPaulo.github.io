## Overview

Autonomous mobile robot system using SLAM for real-time mapping and navigation. Implemented sensor fusion and path planning algorithms in a ROS2 environment on Ubuntu 22.04.

<!-- Add images/GIFs below -->
<!-- ![Demo]({{ site.baseurl }}/assets/img/projects/slam/demo.gif) -->

## Key Features

- Real-time map generation with SLAM
- Sensor fusion (LiDAR + IMU)
- Autonomous path planning and obstacle avoidance
